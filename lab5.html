ЗАВДАННЯ №1 (Варіант 6)

Умова завдання:

Напишіть функцію, яка приймає масив об'єктів і повертає новий масив об'єктів, у якому є знижка на 30% на всі медикаменти, ціна яких перевищує 300 грн. Надайте id для кожного медикамента.

Виконання завдання:

Результат у консолі:

Програмний код:


const medicines = [
    { name: "Noshpa", price: 170 },
    { name: "Analgin", price: 55 },
    { name: "Quanil", price: 310 },
    { name: "Alphacholine", price: 390 },
];

const applyDiscountAndId = (medicines) => {
    return medicines.map((medicine, index) => {
        const newMedicine = {
            id: index + 1,
            name: medicine.name,
            price: medicine.price
        };

        if (newMedicine.price > 300) {
            newMedicine.price = newMedicine.price * 0.7; // Знижка 30%
        }
        return newMedicine;
    });
};

console.log("--- Завдання 1 ---");
console.log("Початковий масив:", medicines);
const discountedMedicines = applyDiscountAndId(medicines);
console.log("Масив зі знижками та ID:", discountedMedicines);

ЗАВДАННЯ №2 (Варіант 6)

Умова завдання:

Відсортуйте масив об'єктів за віком у порядку зростання.

Виконання завдання:

Результат у консолі:

Програмний код:


const users = [
    { name: 'John', age: 27 },
    { name: 'Jane', age: 31 },
    { name: 'Bob', age: 19 },
];

const sortUsersByAge = (users) => {
    // Створюємо копію масиву, щоб не мутувати оригінальний
    return [...users].sort((a, b) => a.age - b.age);
};

console.log("--- Завдання 2 ---");
console.log("Початковий масив користувачів:", users);
const sortedUsers = sortUsersByAge(users);
console.log("Відсортований масив користувачів:", sortedUsers);

ВИСНОВКИ

Під час виконання практичної роботи №5 я здобув практичні навички роботи з об'єктами та масивами об'єктів у JavaScript. Я навчився використовувати перебираючі методи масиву, такі як map() для трансформації даних (додавання полів, зміна значень за умовою) та sort() для впорядкування об'єктів за певним критерієм.

Також я закріпив знання щодо використання стрілочних функцій та колбеків, що дозволяє писати більш лаконічний та сучасний код. Виконання завдань допомогло зрозуміти, як ефективно маніпулювати структурами даних для вирішення типових задач фронтенд-розробки.
