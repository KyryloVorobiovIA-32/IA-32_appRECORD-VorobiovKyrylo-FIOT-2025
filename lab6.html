Програмний код:


// HTML розмітка списку та кнопки
// 
     
  • 1
    •  ... 

// 

function doubleValues() {
    // Отримуємо всі елементи li у списку
    const listItems = document.querySelectorAll('#values-list li');

    // Проходимо по кожному елементу
    listItems.forEach(item => {
        // Отримуємо поточне числове значення
        const currentValue = parseInt(item.textContent);
        // Множимо на 2 та оновлюємо текст
        if (!isNaN(currentValue)) {
            item.textContent = currentValue * 2;
        }
    });
}

ЗАВДАННЯ №2: Делегування подій (Сайт)

Умова завдання:

У власному WEB-застосунку створити програмний код делегування подій: при виборі елемента в колекції елементів - повинне відкритись модальне вікно, яке містиме опис цього елемента.

Що було зроблено:

На сторінці "Типи чашок" (`Types.html`) реалізовано галерею зображень. До кожного зображення додані data-атрибути з інформацією (назва, рік, опис). Використано делегування подій: один слухач `click` додано на батьківський контейнер галереї. При кліку на будь-яку картку зчитуються дані з її атрибутів і відкривається модальне вікно з детальною інформацією.

Реалізація (HTML - Types.html):


<!-- Контейнер для делегування -->
<div class="types-container" id="types-gallery">
    <div class="type-wrapper">
        <!-- Елемент з даними -->
        <div class="type-image"
             data-title="Керамічна чашка"
             data-year="близько 24 000 років до н.е."
             data-info="...">
            <img src="../img/ceramica.jpg" alt="Кераміка">

Реалізація (JS - scripts.js):


function setupModalDelegation() {
    const galleryContainer = document.getElementById('types-gallery');
    const modal = document.getElementById('cupModal');

    // Делегування: слухач на контейнері
    galleryContainer.addEventListener('click', function(event) {
        // Перевірка, чи клік був по картці
        const targetCard = event.target.closest('.type-image');
        if (!targetCard) return;

        // Отримання даних
        const title = targetCard.getAttribute('data-title');
        const year = targetCard.getAttribute('data-year');
        const info = targetCard.getAttribute('data-info');

        // Заповнення та відкриття модалки
        document.getElementById('modalTitle').textContent = title;
        document.getElementById('modalYear').textContent = year;
        document.getElementById('modalInfo').textContent = info;
        modal.style.display = 'block';
    });

    // Логіка закриття модалки...
}
